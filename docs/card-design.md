# 卡牌设计

## 卡牌标签

每张 `CardData` 都有可多选的卡牌标签，用于词条生效检索、流派分类和后续奖励筛选。

当前支持的标签：

| 标签 | 用途               |
|----|------------------|
| 仪式 | 用于仪式相关词条、效果和流派检索 |
| 法术 | 用于法术相关词条、效果和流派检索 |
| 战意 | 用于战意相关词条、效果和流派检索 |

代码中可通过 `CardData.HasTag()`、`CardData.HasAnyTag()`、`CardData.HasAllTags()` 判断卡牌标签。

## 生效位置

每张 `CardData` 都有生效位置配置，用于控制卡牌放在 1-5 号槽位中的哪些位置时会执行效果。

- 默认值为「任意位置」，兼容旧卡牌配置。
- 可配置 1-5 号位的任意组合。
- Odin Inspector 中提供「任意位置」「奇数位」「偶数位」快捷按钮，便于配置常用模式。
- 卡牌放在未配置的槽位时，仍会在结算时进入弃牌堆，但不会触发卡牌效果、卡牌印记或槽位印记。
- 槽位背景颜色用于提示生效状态：空槽为灰色，位置有效为绿色，位置无效为红色。

常用配置：

| 配置   | 对应槽位      |
|------|-----------|
| 任意位置 | 1、2、3、4、5 |
| 奇数位  | 1、3、5     |
| 偶数位  | 2、4       |

## 增伤效果

`BoostSlotCardEffectSO` 用于提高指定槽位的后续主卡牌效果数值，可配置目标槽位和提升方式。

目标槽位支持 1-5 号位任意组合，也可以直接选择任意槽位、奇数槽位或偶数槽位。

当前支持的提升方式：

| 方式    | 说明            |
|-------|---------------|
| 固定增加  | 在原始效果数值上增加固定值 |
| 百分比增加 | 按百分比提高原始效果数值  |
| 倍率提升  | 将原始效果数值乘以指定倍率 |

增伤只在本轮槽位结算期间存在。若增伤卡位于目标槽位之前，会影响目标槽位后续结算的主卡牌伤害和治疗数值；若目标槽位已经结算完成，则不会回溯生效。
